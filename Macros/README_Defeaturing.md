# Defeaturing マクロ（穴・フィレット一括削除）

## ファイル
- `DefeaturingHolesAndFillets.FCMacro`

## 機能
- **穴**: 直径 6.5 mm 以下の円筒面を削除（半径 3.25 mm 以下）
- **フィレット**: R 4 mm 以下のトーラス面を削除

対象オブジェクトを選択してマクロを実行すると、元オブジェクトはそのままで、穴とフィレットを削除した新しいオブジェクト「Defeatured」が作成されます。

## 使い方
1. FreeCAD で Part ワークベンチに切り替え
2. ツリーで対象オブジェクト（STEP など）を **1つ** 選択
3. **マクロ → マクロの実行** で `DefeaturingHolesAndFillets.FCMacro` を選択して実行  
   または **マクロ → マクロのダイアログ** で「作成」→ このファイルを指定してツールバーに登録し、実行

## 設定の変更
マクロ先頭の定数で変更できます。
- `HOLE_MAX_RADIUS_MM` … 穴とみなす円筒の最大半径（mm）。デフォルトは 3.25（直径 6.5 mm）
- `FILLET_MAX_RADIUS_MM` … フィレットとみなすトーラスの最大 R（mm）。デフォルトは 4.0

## 注意
- 複数ソリッドの形状は、ソリッドごとにデフィーチャリングしてから Compound で結合します。
- 削除対象が1面もない場合は「削除対象の穴・フィレット面は見つかりませんでした」とメッセージが出て終了します。
- 幾何的に削除できない組み合わせの場合は、そのソリッドだけスキップされ警告が出ます。
