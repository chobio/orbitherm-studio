# 表示パラメータ メモ

ThermalAnalysis ワークベンチで使用している表示まわりの定数・仕様のメモです。

**設定の変更**: メニュー「表示・伝導パラメータ設定」から、以下で挙げるノード表示・コンダクタンス線・伝導コンダクタンスのペア選定パラメータを GUI で変更できます。保存先は FreeCAD のユーザー設定 `Mod/ThermalAnalysis` です。

## ノード表示

| 項目 | 値 | 場所（core.py） |
|------|-----|-----------------|
| デフォルト PointSize（共有ノード・面ごとノード） | 14 | 共有ノード・面ノード作成時 |
| デフォルト PointSize（分割サブノード） | 10 | 分割面ノード作成時 |
| ノード球サイズ（面基準 100%時） | 面対角線の 12% | `_RA_NODE_SPHERE_FRACTION_FACE = 0.12` |
| ノード球サイズ（全体基準 100%時） | 全体対角線の 3% | `_RA_NODE_SPHERE_FRACTION_GLOBAL = 0.03` |
| 球半径の最小/最大 [mm] | 0.5 / 500.0 | `_RA_NODE_SPHERE_RADIUS_MIN_MM`, `_RA_NODE_SPHERE_RADIUS_MAX_MM` |
| 表示設定でノードサイズ適用時の PointSize 係数 | radius_mm / 1.2 | `run_set_node_point_sizes` 内 |

## コンダクタンス線表示

| 項目 | 値 | 場所（core.py） |
|------|-----|-----------------|
| 線の太さ [px] | 2.5 | `_RA_CONDUCTANCE_LINE_WIDTH` |
| 伝熱コンダクタンスの色 | 青 (0.2, 0.5, 1.0) | `_RA_CONDUCTION_LINE_COLOR` |
| 輻射コンダクタンスの色 | オレンジ (1.0, 0.35, 0.0) | `_RA_RADIATION_LINE_COLOR` |

適用タイミング: 伝熱/輻射コンダクタンス計算でリンク作成時、および手動で伝熱コンダクタンス追加時。  
既存リンクは再計算すると新しい太さ・色が反映される。

## ホバー表示（面・ノード番号）

- 設定: ユーザー設定 `Mod/ThermalAnalysis` の `HoverLabelEnabled`（デフォルト True）、`HoverThrottleMs`（デフォルト 100）
- メニュー: 「表示」→「面・ノード番号のホバー表示」で ON/OFF
- 実装: `InitGui.py` の `_hover_callback`、`getObjectInfo` の引数はタプルと 2 引数の両対応。オブジェクト名は `Object` / `ObjectName` / `PickedObject` / `Name` をフォールバック。アクティブドキュメント切り替え時に `_HoverDocObserver` でホバーを再登録。

## 伝導コンダクタンス（異なる面同士）のペア選定

- **仕様**: 共有エッジを持つ2面について、そのエッジから **5 mm 以内** にあるノードを「稜線上」とみなし、稜線に沿ったパラメータ順で 1 対 1 にペアリングして、ペアごとに 1 本の伝熱コンダクタンスリンクを作成する。
- **フォールバック**: 稜線付近にノードが一つも無い場合は、従来どおり「代表ノード 1 つ × 代表ノード 1 つ」で 1 本だけ作成する。
- **手動で指定したい場合**: メニュー「伝熱コンダクタンスを追加」で、2 つのノードを選択してから実行すると、その 2 ノード間だけに手動リンクを追加できる。自動計算で意図しないペアになる面同士は、一度自動計算せず、必要なノード同士を手動で追加する運用が可能。
